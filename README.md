# Rockchip 平台设备

## 支持的设备列表（Lede)
- Orange Pi R1 Plus LTS (Lede & ImmortalWrt)
- Orange Pi R1 Plus
- ...
- FriendlyArm NanoPi R2S


## 注意事项
 - ImmortalWrt的**Orange Pi R1 Plus LTS** 只编译了 **1.5 GHz** 主频版本。
 - Lede的**Orange Pi R1 Plus LTS** 编译了两个主频的版本。如果您在使用 **1.6 GHz** 版本时遇到启动失败的问题，可以尝试切换到 **1.5 GHz** 版本。
